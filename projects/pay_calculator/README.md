# Installation

```shell
mkdir pay_calculator
cd pay_calculator
npm create react-app . --template typescript
```

# Development

```shell
npm start
```

# APP URL

[Paycalculator AU](https://pay-calculator-au.netlify.app/)

# Update packages

```shell
npm outdated
# npm update will not update the major versions, only minor and patch (major.minor.patch)
npm update
```
